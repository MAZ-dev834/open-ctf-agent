# Skill Evolution Notes

这份文件在公开版中只保留通用演进原则，不再记录真实题目、历史 flag 或个人工作区路径。

## 演进原则

- 每次解题后只沉淀可复用的方法，不沉淀具体赛事私有数据。
- 记录的内容应是“攻击面识别方法”“证据组织方式”“失败分支关闭标准”，而不是题目答案。
- 对新模板的收录标准：
  - 能复现
  - 能解释为什么有效
  - 至少在一个挑战上证明过有价值
- 对旧模板的淘汰标准：
  - 只对单题有效
  - 依赖作者本地环境
  - 无法给出明确输入/输出契约

## 记录格式

建议每条演进记录只包含：

- 适用题型
- 触发信号
- 推荐动作
- 常见误判
- 最小验证步骤
